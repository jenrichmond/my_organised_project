
# Thesis Project Folder Template

This repository contains a ready-made folder structure for organising your thesis project. Instead of building it yourself from scratch, you can use this as a starting point and spend your energy on the research.

## How to get it

1.  Click the green **Code** button at the top of this page
2.  Select **Download ZIP**
3.  Unzip the file on your computer (put it in a place that is backed up automatically)
4.  Rename the folder to match your project (e.g., `Honours_2026_YourName`)

## What's inside

Rather than keeping everything in a single folder, it is a good idea to organise your research files in a structured way. 


![](thesis_folder.png)

Each folder contains a readme.txt explaining what goes in it.

## How to use it

-   Put files in the folder that matches what they are
-   Use consistent file names (e.g., `2026-06-01_MeetingNotes.docx` or `Draft_01.docx`)
-   When a file is replaced by a newer version, move the old one to `08_Archive/` rather than deleting it
-   Never edit anything inside `04_Data_Processing/Raw_Data/` — that folder is for original data files only

## Questions?

If you're not sure where something goes, the readme.txt inside each folder will help. When in doubt, make a decision and be consistent — a good-enough system you actually use beats a perfect system you don't.
